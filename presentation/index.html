1. Introduction

Image Denoising

  • Remove noise from images captured by smartphone sensors
  • Noise is spatially varying and signal-dependent
  • SIDD (Smartphone Image Denoising Dataset) — the standard benchmark

Competition

  • Kaggle SIDD Benchmark — sRGB PSNR
  • Metric: PSNR (Peak Signal-to-Noise Ratio), higher is better
  • Test set: 1280 blocks, each 256×256×3 sRGB
  • Gold threshold: ≥ 40.0019 PSNR

3. Methodology — NAFNet

Nonlinear Activation Free Network (ECCV 2022)

  • Key innovation: Remove ReLU/GELU/Sigmoid from direct signal path
  • Use SimpleGate(x) = LayerNorm(x₁) · x₂ instead
  • Architecture: width=64, middle_blks=12, enc_blks=[2,2,4,4], dec_blks=[4,4,2,2]
  • Pre-trained on SIDD: NAFNet-SIDD-width64.pth (464 MB)

3. Methodology — Restormer

Efficient Transformer for Image Restoration (CVPR 2022)

  • Key innovation: Multi-scale GFN blocks with channel attention (not spatial)
  • Progressive architecture: coarse → refined at multiple scales
  • More efficient than spatial attention for high-res images
  • Pre-trained on SIDD: Restormer_SIDD.pth (310 MB)

3. Methodology — 8-way TTA

Test-Time Augmentation

  • Apply 8 geometric transformations to each input block
  • 4 rotations: 0°, 90°, 180°, 270° × 2 flips: none, horizontal flip
  • Run model on each → apply inverse transform → average predictions
  • Key lesson: Inverse transforms must be correctly applied!

Memory Optimization

  • Restormer 8-way TTA exceeds 12GB VRAM at batch=8
  • Solution: batch_size=1, sequential augmentation
  • Inference time: ~80 min for 1280 blocks (vs ~20 min without TTA)

4. Ensemble Strategy

Weighted Average of Two Models

result = α × NAFNet-TTA + (1-α) × Restormer-FT-TTA

Why NAFNet + Restormer?

  • Complementary error characteristics:
    • NAFNet: local convolution, captures local texture patterns
    • Restormer: global channel attention, captures global structure
  • Equal-weight ensemble (α=0.5) scored only 39.95 — weighted is critical!

⚠️ Pairing TTA+TTA (NAFNet-TTA + Restormer-TTA) severely degrades to ~39.82 — TTA errors compound!

5. Key Findings

  1. Optimal ensemble: α=0.745 (NAFNet-TTA weight) → 40.4852 PSNR
  2. Don't pair TTA variants: NAFNet-TTA + Restormer-TTA = 39.82 (severe degradation)
  3. Non-TTA NAFNet + Restormer-TTA is the correct pairing
  4. 3-model ensemble failed: Restormer-TTA + Restormer-MS-TTA are too correlated
  5. Weighted ensemble is critical: Equal-weight = 40.26, weighted = 40.49
  6. Fine-tuning Restormer gave no improvement (40.0899 vs 40.0902 pre-trained)
  7. Gold medal achieved: 40.4852 vs 40.0019 threshold (+0.48 dB)

6. Future Work

  • SOTA models: Try HAT (2024), CGNet — potential to exceed 40.6 PSNR
  • Multi-scale TTA: Resize to 0.9x, 1.0x, 1.1x — more diverse augmentation
  • Truly diverse 3-model ensemble: NAFNet + Restormer + MIRNetv2 (with correct pairing)
  • Training data: Fix SIDD Medium sRGB download for fine-tuning on task-specific data
  • Better fine-tuning: Lower LR, more augmentation, more epochs

7. Conclusion

  • Built NAFNet and Restormer inference pipelines on SIDD benchmark
  • Optimal weighted ensemble achieves 40.4852 PSNR (α=0.745)
  • Exceeds gold medal threshold by +0.48 dB
  • Key insight: Don't pair TTA outputs — use NAFNet-TTA + Restormer-TTA (non-TTA NAFNet is the buffer)
  • 3-model ensembles only work with truly architecturally diverse models
  • Equal-weight ensembles significantly degrade performance
